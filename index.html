DURAMENU v1.0

Responsive Bootstrap Menu

Duramenu is a Bootstrap 3.X powered responsive navigation system

IE 9+, Chrome 40+, Mozilla 35+, Opera 30+

Aligned Content

Content is neat

With tabbed menus, content is well organized

Duramenu comes with dropdown-tabbed and dropdown-tabbed-mega classes. Both can be used and adapted easily.

Details can be found in our Documentation

Fully Responsive & Mobile Optimized

On mobile devices, duramenu is touch-enabled

For larger devices, duramenu works in :hover state. However, if the screen size is smaller, Duramenu behaves as a Bootstrap click-based dropdowns

To see, how it works, you can use the following QR-code

Color Options

10 predefined themes, easy to modify

You can also edit predefined themes, or create your own easily

Use settings button to see color themes live

Boxed or Full-Width Layout

How do you want to design?

Depending on your webpage design, duramenu can either be used as Boxed or Full-width

By default, duramenu is boxed. However, it can be easily transformed into full-width menu by including the fullwidth css file provided

Various examples can be found in the download package when you purchased the item

Product Carousel

Bootstrap product carousel in your navigation

Just the same as Bootstrap 3.X carousel usage

No need for an external javascript library
